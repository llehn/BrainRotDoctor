How it works

You make rules. A rule is simply what to block and when.

🎯

Pick what to block

Choose from a catalog — Instagram Reels, YouTube Shorts, TikTok, X, Reddit and more — or add any custom site. No hosts or regex to fiddle with.

⏳

Allowance per hour

"Allow 5 minutes per hour, then block." Enough for a peek and a laugh, not enough to disappear for the afternoon. Refills every hour.

🌙

Or a hard schedule

Block completely during a window — say 23:00–07:00 on weekdays — so bedtime stays bedtime and mornings start without the feed.

🔒

Strict mode

Lock everything for a set time behind a double opt-in. While it's on you can't edit the rules, quit, or even uninstall from Windows the easy way.

Your rules at a glance — with live status: how much time is left, or what's blocked right now.

Install in three clicks

1

Download

Grab the single BrainRotBlocker.exe and double-click it.

2

Get past SmartScreen

Windows will warn about an unknown publisher (it's unsigned, open-source). Click More info → Run anyway.

3

Install

Click Install in the prompt. It sets itself up for your user, starts with Windows, and begins protecting you.

No administrator rights required — it installs under your own user folder. To remove it later, use Windows "Apps & features" (this won't work while strict mode is active — that's the point).

The details

Closes the tab, not the browser. It watches the active tab via Windows UI Automation and closes only the offending page.
Keeps the useful parts. Block the reels and the feed while DMs and stories stay reachable.
Two-process guard. A paired watchdog restarts the app instantly if it's killed — no quick-quit escape.
Survives restarts. Registers per-user autostart, so a reboot doesn't get you off the hook.
31 languages. Follows your Windows language automatically, switchable in Settings.
Light & dark. A clean, modern interface that follows your system theme.
No account, no cloud. Everything stays on your machine. Free and open source.
Strict-mode-aware uninstall. The easy uninstall is blocked while you're committed — deliberate friction, by design.
